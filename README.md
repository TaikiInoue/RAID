## 4. Usage
```
git clone git@github.com:TaikiInoue/RIAD.git
cd RIAD
```

```
docker pull taikiinoue45/mvtec:riad
```

```
docker run --runtime nvidia -it --rm taikiinoue45/mvtec:riad /usr/bin/zsh
```

```
sh run.sh
```

```
mlflow ui
```
